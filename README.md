# AI Mock Interview Simulator

A comprehensive platform for simulating mock interviews using AI technologies. This project provides an interactive environment for users to practice interviews, receive feedback, and improve their skills.

## Features

- AI-powered interview questions
- Real-time feedback and evaluation
- Resume analysis
- Behavioral and technical assessment
- Adaptive questioning based on user performance

## Setup

1. Clone the repository
2. Navigate to the backend directory
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: `./start.sh`
